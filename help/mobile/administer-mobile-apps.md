# Aplicaciones móviles{#mobile-apps}

{{ue-over-mobile}}

Adobe Experience Manager AEM () Mobile forma parte de la plataforma digital multicanal de Adobe Experience Manager que también utiliza AEM Assets, Sites, Screens, Workflow, User Manager, Commerce y Forms. AEM Mobile ofrece una notable flexibilidad de implementación, lo que permite a las empresas utilizar una variedad de fuentes de contenido.

AEM Mobile ofrece varios métodos para crear aplicaciones. Las diferentes formas de crear aplicaciones de son las siguientes:

* [Administración de contenido para AEM Mobile On-demand Services](/help/mobile/aem-mobile.md)
* [Administración de contenido para Adobe PhoneGap AEM Enterprise con el servicio de administración de](/help/mobile/administer-phonegap.md)

>[!NOTE]
>
>Si no tiene experiencia previa con Adobe Experience Manager AEM, puede que quiera seguir [Introducción a la administración de datos](/help/sites-deploying/deploy.md) para los usuarios de la aplicación .
