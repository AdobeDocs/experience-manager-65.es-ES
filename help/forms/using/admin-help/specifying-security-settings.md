# Especificación de la configuración de seguridad {#specifying-security-settings}

Forms permite controlar si las entidades externas de las entradas XML se resuelven. De forma predeterminada, se resuelven, pero puede cambiar este comportamiento para aumentar la seguridad del sistema de formularios AEM.

**Impedir el procesamiento de archivos de datos XML que contienen referencias a entidades externas**

1. En la consola de administración, haga clic en **[!UICONTROL Servicios > Formularios]**.
1. Desactive la casilla de verificación Resolver entidades externas.
1. Haga clic en **[!UICONTROL Guardar]**.
